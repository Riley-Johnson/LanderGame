Variables

Variables let you remember values between timesteps (frames).

Creating Variables

  1. Click "Variables" in the toolbox
  2. Click "Create variable"
  3. Give it a name (e.g., "landing_mode")

Using Variables

Variables persist between timesteps, so you can:

  • Count how many times something happened
  • Remember if you've entered a landing phase
  • Track time elapsed
  • Store calculated values

Example

Track landing sequence:

If altitude < 100 and landing_mode = 0:
  set landing_mode to 1
  print "Starting landing sequence"
