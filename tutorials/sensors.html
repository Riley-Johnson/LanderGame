Sensor Blocks

Sensors give you information about your lander's current state:

Position Sensors

  • get altitude - Height above ground in meters
  • get horizontal position - Distance from center (x=0)

Velocity Sensors

  • get velocity - Vertical speed (negative = falling)
  • get horizontal velocity - Sideways speed

Rotation Sensors

  • get angle - Rotation in radians (0 = pointing up)
  • get angular velocity - How fast you're rotating (rad/s)

Example

Use sensors in if-blocks to control your lander:

If altitude < 50, then set throttle to 0.8
