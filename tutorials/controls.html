Control Your Lander

Throttle Control

set throttle - Controls main engine power (0.0 to 1.0)

  • 0.0 = engine off
  • 0.5 = half power
  • 1.0 = maximum thrust

Careful, the main engine will use up fuel over time.

Rotation Controls

set RCS - Controls RCS thruster power (-1.0 to 1.0)

  • -1.0 = full power counterclockwise (CCW)
  • 0.0 = no rotation
  • 1.0 = full power clockwise (CW)

RCS (Reaction Control System) thrusters help you orient the spacecraft and currently don't use any fuel. The magnitude controls thrust strength, allowing for precise rotation control.
